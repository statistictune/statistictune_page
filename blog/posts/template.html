Sobrevivência Kaplan-Meier
28 Mai 2026 · Rafael Aguilar Magalhães · 8 min de leitura

Introdução à Análise de Sobrevivência:
do conceito ao Kaplan-Meier

O que é censura, por que dados de tempo-até-evento precisam de métodos especiais e como interpretar uma curva de Kaplan-Meier passo a passo — do zero ao gráfico publicável em R.

Por que métodos especiais?

Em estudos clínicos, frequentemente o desfecho de interesse é o tempo até um evento — morte, recaída, hospitalização, progressão de doença. O problema: nem todos os participantes experimentam o evento durante o período de acompanhamento. Alguns saem do estudo, outros simplesmente não chegaram ao evento quando a coleta encerrou.

Esse fenômeno se chama censura. Ignorar pacientes censurados seria desperdiçar informação valiosa; tratá-los como se tivessem experimentado o evento seria mentira estatística. É exatamente para isso que existem os métodos de análise de sobrevivência.

S(t) = P(T > t) = 1 − F(t)
S(t) = função de sobrevivência; T = tempo até o evento; F(t) = função de distribuição acumulada.

O estimador de Kaplan-Meier

O estimador de Kaplan-Meier (KM) é o método não-paramétrico padrão para estimar a função de sobrevivência a partir de dados com censura. A ideia central é atualizar a probabilidade de sobrevivência apenas nos tempos em que eventos ocorrem, ignorando os instantes de censura.

Atenção: O estimador KM assume que a censura é não-informativa — ou seja, o fato de um paciente sair do estudo não deve estar relacionado ao seu prognóstico. Violar essa suposição invalida a estimativa. 
# Instalar se necessário
# install.packages("survival")
# install.packages("survminer")

library(survival)
library(survminer)

# Ajustar curva KM
km_fit <- survfit(Surv(time, status) ~ group, data = seu_dataset)

# Visualizar
ggsurvplot(km_fit,
           data       = seu_dataset,
           pval       = TRUE,
           conf.int   = TRUE,
           risk.table = TRUE,
           palette    = c("#1B6B7B", "#C9A84C"),
           xlab       = "Tempo (dias)",
           ylab       = "Probabilidade de sobrevivência")
